How does it work?

Web application that scrapes job information from your LinkedIn using Prospeo API and scoures the internet for current openings using Metaphor API. Jobbly then uses OpenAI's LLM API to determine which jobs are the most relevant to your past experiences and skills. It then displays the openings, where to find them, and why the AI thinks you are a match!

No more applying to hundreds of job openings that don't fit your skills and experiences; Jobbly does all the hard work for you!

Why and How did I do it?

I have always struggled with applying to jobs that are relevant to my skills and past experiences. There are so many out there and difficult to find. Jobbly was my solution! Jobbly is my interview project for the company Metaphor, where the assignment was to create something using Metaphor API within 3 days, taking no more than 1-3 hours.

My project took roughly 2.5 hours from start to finish and I did it one sitting on Oct 1st Sunday Night 2023.
